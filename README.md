# 📱 Calculadora Expo

Aplicativo de calculadora simples desenvolvido com **React Native + Expo**, com interface inspirada na calculadora do iPhone.

## 🚀 Tecnologias utilizadas

* React Native
* Expo
* React Native Paper
* JavaScript
* Pressable Components

## 📋 Funcionalidades

* Adição (+)
* Subtração (-)
* Multiplicação (×)
* Divisão (÷)
* Botão de limpar (C)
* Botão de apagar (⌫)
* Suporte a números decimais
* Interface minimalista

## 🎨 Interface

O aplicativo possui:

* Tema escuro
* Botões arredondados
* Operadores em destaque na cor laranja
* Display responsivo
* Design inspirado na calculadora do iPhone

## ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/joaodagostin/calculadora-expo.git
```

Entre na pasta do projeto:

```bash
cd calculadora-expo
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npx expo start
```
